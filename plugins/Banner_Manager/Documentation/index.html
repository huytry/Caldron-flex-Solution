Banner Manager
plugin for
RISE CRM

Banner Manager is a dashboard banner management plugin. It can help you to increase satisfaction and inspiration of the team members and clients. It requires RISE v3.0 or higher.

Installation
  • Please login as Admin in your RISE app.
  • Go to the Settings > Plugins page.
  • Install the Banner-Manager-for-RISE-CRM.zip which you have downloaded from the codecanyon.
    It'll require the item purchase code. Don't use any nulled version for your security.
  • If everything is OK, you'll get a success message after installation.
  • Activate the plugin from the option dropdown of the plugins list.
  • Go to Settings > Setup > Banner Manager.
  • You can choose non-admin users to manage banners. Admin users will always get access.
  • Then the permitted users will get a new menu item in the left menu. Labeled as Banner Manager.
    Please note that, if the user's left menu or default left menu is already edited, then the Banner Manager won't be shown by default. Then you can add the menu manually from the left menu settings.
  • After creating any banner, that'll be shown in the selected user(s) dashboard on the certain date(s).

Upgrading Guide

There is a easy way to update the plugin.

To install any updates, please go to Settings > Plugins > Banner Manager > Dropdown > Updates. The download will be started automatically if there is any new update available. Once the download is complete you can install the updates by clicking on the link.

Please backup all files and database before starting the installation of new updates.
